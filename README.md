# Basit Harcama Takip Uygulaması (Python)

Terminal üzerinden çalışan basit bir harcama takip uygulaması.

## Özellikler

- Yeni harcama ekleme (açıklama, tutar, kategori)
- Eklenen harcamaları listeleme
- Toplam harcamayı hesaplama
- Hatalı tutar girişinde kullanıcıyı uyarması

## Kullanılan Teknolojiler

- Python 3
- Temel veri yapıları (liste, sözlük)
- Döngüler, koşullar, fonksiyonlar

## Çalıştırma

```bash
python expense_tracker.py
